# 📊 Marketing Campaign Analytics — PySpark & SQL

## 📌 Project Overview
Large-scale marketing campaign analytics pipeline
built using PySpark and SQL. Processes 1M+ records
to uncover trends, improve campaign ROI, and deliver
actionable insights through Tableau dashboards.

## 🏗️ Architecture
Raw Campaign Data → PySpark ETL → Data Warehouse
→ SQL Analytics → Tableau Dashboards → Stakeholders

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| PySpark | Large Scale Data Processing |
| Python | Pipeline Development |
| SQL | Query Writing & Optimization |
| Tableau | Dashboard & Visualization |
| Advanced Excel | Reporting & Analysis |
| Git | Version Control |
